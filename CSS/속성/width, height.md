# width
- `width` 속성은 요소의 가로 크기를 지정하기 위해 사용한다.
- 값
  - 전역 값
  - `[크기]`

# height
- `height` 속성은 요소의 세로 크기를 지정하기 위해 사용한다.
- 값
    - 전역 값
    - `[크기]`

# max-width
- `max-width` 속성은 요소가 가질 수 있는 최대 가로 크기를 지정하기 위해 사용한다.
- 값
  - 전역 값
  - `[크기]`
